# SteerIn 🚗

**Kelola kendaraan pribadimu dengan mudah, aman, dan sepenuhnya milikmu.**

SteerIn adalah aplikasi Android untuk mengelola mobil dan motor — dari perawatan berkala, riwayat servis, pelacakan perjalanan, hingga penemuan bengkel terdekat. Login diperlukan untuk memakai aplikasi, sementara data kendaraan tetap diprioritaskan tersimpan lokal di perangkatmu.

## 🚀 Fitur Utama

### 🏠 Multi-Vehicle Garage
Kelola beberapa kendaraan (mobil & motor) dalam satu garasi. Tambah foto, data spesifikasi, dan pantau odometer masing-masing.

### 🔧 Maintenance Tracking
Catat setiap servis: ganti oli, rem, ban, filter, dan lainnya. SteerIn menghitung kapan servis berikutnya berdasarkan interval kilometer atau waktu.

### 📍 GPS Trip Logging
Rekam perjalananmu secara otomatis. Lihat rute, jarak, dan durasi perjalanan.

### 🗺️ Workshop Discovery
Temukan bengkel terdekat di peta. Lihat jam buka, status (buka/tutup), dan lokasi langsung dari aplikasi.

### 📊 Vehicle Health Score
SteerIn menganalisis kondisi kendaraanmu berdasarkan usia, odometer, dan riwayat perawatan. Dapatkan skor kesehatan dan rekomendasi.

### 🔍 Diagnostic Engine
29 aturan diagnosa untuk mengecek komponen kendaraan. Ekspor hasil ke JSON atau bagikan ke mekanik.

### 📸 Odometer OCR
Foto odometer, SteerIn otomatis membaca angkanya dengan ML Kit. Tidak perlu input manual.

### ☁️ Cloud Sync & Backup
Backup data terenkripsi ke cloud. Restore kapan saja. **Fitur opsional** — login diperlukan, tetapi data kendaraan tetap diprioritaskan tersimpan lokal.

### 🔒 App Lock
Kunci aplikasi dengan biometrik (sidik jari/face unlock) untuk menjaga privasi data kendaraanmu.

## 🔐 Privasi & Keamanan

SteerIn dibangun dengan prinsip **privacy-first**:

- **🏠 Local-First** — Login diperlukan, namun data kendaraan tetap diprioritaskan tersimpan di perangkatmu
- **🚫 No Ads, No Trackers** — Tidak ada iklan, tidak ada pelacakan
- **🔒 Enkripsi Backup** — Backup dienkripsi dengan PBKDF2-SHA256
- **🔐 Biometric Lock** — Kunci aplikasi dengan sidik jari atau face unlock
- **📤 Ekspor Data** — Ekspor semua data kapan saja dalam format standar

Baca [Kebijakan Privasi](https://steerin-site.vercel.app/privacy-policy) lengkap.
